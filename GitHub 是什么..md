一、GitHub 是什么
 
GitHub 是全球最大的基于 Git 的代码托管与协作平台（2018 年被微软收购） 。
 
- Git：本地代码版本管理工具（记录修改、分支、回滚）
​
- GitHub：云端平台，把 Git 管理的代码放到网上，实现共享、协作、开源、备份
 
二、核心用途
 
- 📦 代码托管/备份：免费存公开/私有仓库，不怕本地丢失
​
- 👥 团队协作：多人同时开发、代码审查（PR）、分支管理、冲突解决
​
- 🧑‍💻 开源社区：看 React、Vue、TensorFlow 等顶级项目源码；提交 PR 贡献代码
​
- 📝 个人技术名片：用 GitHub Pages 做博客/作品集，求职加分
​
- 🤖 自动化（CI/CD）：GitHub Actions 自动测试、构建、部署
​
- 🧩 项目管理：Issues 追踪 Bug/需求、Projects 看板、Wiki 文档
 
三、新手快速上手（3 步）




# 克隆仓库到本地
git clone https://github.com/用户名/仓库名.git

# 提交代码
git add .
git commit -m "提交说明"
git push origin main



四、核心概念（必懂）
 
- Repository（Repo）：项目仓库（文件夹）
​
- Branch（分支）：独立开发线，不影响主代码
​
- Pull Request（PR）：提交代码合并请求，用于审查
​
- Issue：Bug/需求/任务跟踪
​
- GitHub Pages：免费静态网站托管（博客/简历）
